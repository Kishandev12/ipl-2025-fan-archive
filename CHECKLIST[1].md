## Step-by-Step Deployment

### 1. Create Repository
- [ ] Go to https://github.com
- [ ] Click "+" ‚Üí "New repository"
- [ ] Name it: `ipl-2025-fan-archive`
- [ ] Make it **Public**
- [ ] Check "Add a README file"
- [ ] Click "Create repository"

### 2. Upload Files
- [ ] Click "Add file" ‚Üí "Upload files"
- [ ] Drag all 4 files into the upload area
- [ ] Write commit message: "Initial upload of IPL 2025 Fan Archive"
- [ ] Click "Commit changes"

### 3. Enable GitHub Pages
- [ ] Click "Settings" in your repository
- [ ] Click "Pages" in left sidebar
- [ ] Under "Source", select:
  - Branch: `main`
  - Folder: `/ (root)`
- [ ] Click "Save"

### 4. Wait for Deployment
- [ ] Wait 1-2 minutes
- [ ] Refresh the Settings ‚Üí Pages page
- [ ] Copy your website URL: `https://YOUR-USERNAME.github.io/ipl-2025-fan-archive/`

### 5. Test Your Website
- [ ] Open the URL in your browser
- [ ] Test navigation (Headlines, Reddit, Memes, etc.)
- [ ] Click on 2-3 external links (they should open)
- [ ] Click "About Omeka" link (should load second page)
- [ ] Test on mobile phone (should be responsive)

## Common Issues & Fixes

### Issue: 404 Error Page
- [ ] Check that main file is named `index.html` (not `ipl_2025_archive_enhanced.html`)
- [ ] Verify GitHub Pages is enabled in Settings ‚Üí Pages
- [ ] Wait 2-3 more minutes (sometimes takes time)

### Issue: Omeka Link Broken
- [ ] Verify `ipl_2025_omeka_connection.html` is uploaded
- [ ] Check filename spelling exactly matches
- [ ] File names are case-sensitive!

### Issue: Graphics Not Showing
- [ ] Try a different browser (Chrome, Firefox, Safari)
- [ ] Clear browser cache (Ctrl+Shift+R or Cmd+Shift+R)
- [ ] SVGs are embedded, should work automatically
